## 🌐 Website content workflow

The website is intentionally data-driven. Its announcements, course guide, and curated study links come from [`data/site-data.js`](data/site-data.js), so useful information can be added without editing the page layout or JavaScript logic.

- Add an official update to `updates`.
- Add or improve a subject entry in `courses`.
- Add a verified learning link to `resources`.

See [`data/README.md`](data/README.md) for the exact format and publishing checklist.
